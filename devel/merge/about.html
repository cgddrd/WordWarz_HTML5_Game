Submission Notes

Please be aware that this game requires the use of HTTP Requests to obtain data files used to generate the internal dictionary.

As a result of security protocols adopted by modern web browsers, this game will not run straight from a local HDD, and will require testing on a webserver.

This game is available to test online at: http://users.aber.ac.uk/clg11/iwp/wordwarz

Game Outline

WordWarz is a web-based game designed to aid in the development of an individual's skills in spelling and typing speed through a fun and challenging medium.

It is built using a combination of the HTML5/Javascript client-side web technologies, and aims to provide an intutive and exciting gameplay via the use of the user's computer keyboard.

Key Features

  • Game mechanics designed to promote the development of spelling and typing speed through the user entering the correct spelling of words as fast as possible.
  • Increasing level difficulty as the user progresses through the game with no explicit ending (i.e. the game continues to become increasingly harder until the user looses all their allocated lives).
  • Randomly generated levels consisting of both enemy, and "powerup" space craft.
  • Support for both keyboard and mouse controls.
  • Collection of sound effects and background music that play throughout the game.
  • Sophisticated movement animation including sprites that automatically turn to face the direction of the player.
  • Pause and muting in-game controls.
  • Collection of in-game acheivements unlocked through play.
  • Local "highest score" display.
  • Word-per-minute score for player calculated in real-time.
